# AI Launcher MVP

An AI-powered Android launcher that understands your context and learns from your feedback.

## Core Concept

Combines the best of three design paradigms:
- **Hero "For You" card** with strong feedback (👍👎)
- **Intent capsules** (Commute / Work / Lunch / Rest) that reshape the home screen
- **Discover Feed** (TikTok-style) with implicit signals + light dismiss
- **User Profile v47** — transparent, editable AI understanding of you

## Status

🏗️ Under active development — auto-built via 5-phase agent pipeline.
